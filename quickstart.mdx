## 1. Get your API Key

To authenticate your requests, you need an API key.

<Steps>
  <Step title="Log in to Dashboard">
    Go to the [Yanna Dashboard](https://app.yanna.pro).
  </Step>
  <Step title="Navigate to Settings">
    Find the API Keys section in your account settings.
  </Step>
  <Step title="Create Key">
    Generate a new API key and copy it. You will need this for all API requests.
  </Step>
</Steps>

## 2. Send a Template

Use the `send` endpoint to generate and send a signing package.

```bash
curl -X POST https://app.yanna.pro/api/v1/api-template/{template_id}/send \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "recipients": [
      {
        "email": "client@example.com",
        "name": "Client Name",
        "role": "signer"
      }
    ],
    "fields": {
      "company_name": "Acme Corp",
      "effective_date": "2023-10-27"
    }
  }'
```
